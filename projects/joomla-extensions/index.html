Joomla Extensions

This page showcases nearly all of the Joomla extensions that I have developed over the years. Please also visit my profile on the Joomla Extensions Directory.

Article Pager

Article Pager for Joomla transforms content management by providing customizable article templates. Ideal for sections like ‘book reviews,’ it allows for consistent layout designs with specific fonts, colors, and arrangements. Create a template once using Joomla’s editor, then simply use tags for variable content like titles and descriptions. Associate each template with an article category, and upon selection, the text auto-fills with the chosen layout. Update the template to change the layout across all linked pages. For inquiries, contact the developer.

Article Tool

Article Tool is a Joomla plugin that enhances articles and K2 articles with additional features. Among the plugin’s capabilities, you can choose to display buttons for printing, downloading (in the PRO version), sharing, and emailing the article. The supported social media icons include Facebook, X, LinkedIn, Pinterest, Instagram, and TikTok. These icons can be placed at the top or bottom of the article. The download function offers two options: creating a PDF displayed in the browser or generating a downloadable PDF. Both printing and PDF creation require no special configurations for individual items; simply activate the plugin. Additionally, you can customize the appearance of printed or downloaded content using style sheets (e.g., font, color, background, line spacing). Article Tool is also compatible with K2. For any issues or suggestions, feel free to contact the developer!

Votes

Votes is a versatile Joomla extension for creating extensive surveys. Unlike standard surveys with limited options, Free Votes allows users to rate each answer on a scale, offering deeper insights into varied responses. It includes a feature for users to add and vote on their own answers, enhancing interactivity. Administrators receive notifications for new user-generated answers. The extension supports multiple voting options and customization with CSS, ensuring adaptability to any site design. Built with HTML5, CSS3, and JavaScript, it’s mobile-friendly and performance-optimized.

Forum translator

Forum Translator is a user-friendly module for Kunena forums, enabling instant post translations. It’s straightforward to set up with no backend configurations—simply activate and it’s ready to go. Users can select their desired language from a dropdown menu, and the module, powered by Google Translate, swiftly translates the content. It utilizes PHP Curl to communicate with Google Translate, ensuring seamless integration and quick translations.

Jcode

Jcode is an innovative Joomla extension that facilitates the insertion of various code types, including HTML, JavaScript, CSS, PHP, AJAX, and Joomla API. It bypasses Joomla’s backend constraints by allowing files to be edited via FTP, providing freedom to use preferred text editors. Additionally, Jcode offers an in-built code editor for on-the-fly editing and is available for free as an open-source resource, making it a versatile tool for developers.

Download

Jexport

JExport is a Joomla component that simplifies the export of site pages into HTML format. It’s particularly useful for static sites without PHP or with outdated PHP versions. JExport allows local Joomla management, enabling changes to settings like maxnestinglevel and maxexecutiontime for exporting HTML pages. Once exported, these pages, along with root folders like images and templates, can be uploaded to the remote server, making JExport ideal for landing pages or infrequently updated sites.

K2 for CB

K2 for CB is a user-friendly plugin for Community Builder that enhances article management. It allows easy editing of graphic settings and assignment of K2 category IDs for articles. Creating a new K2 category with user permissions for article creation and editing is recommended. This integration adds a ‘K2 CB’ tab to the Community Builder profile, offering a comprehensive article editor with image insertion and custom forms, significantly expanding Community Builder’s capabilities.

Map for K2

Map for K2 is a practical plugin for the K2 component in Joomla. It simplifies adding maps to your K2 items by auto-generating markers based on city and address fields, which you create in the K2 backend. Installation is straightforward: set up the plugin and link it to the custom fields. Users can then add new entries with locations, and the map updates automatically—no need to input coordinates manually. This feature is especially beneficial for review sites, allowing users to visually associate products with their geographical locations.

Notice Board for CB

Notice Board is a versatile plugin for Community Builder, enabling a personal message board for each user. It’s functional from version 2.0 and doesn’t require Community Builder. Users can post, view others’ posts, comment, and report rule violations. The editor supports links, images, and videos. Administrators can control post visibility per page and manage email notifications for new posts and comments. It comes with English and Italian language support, with options for customization or adding new languages, making it a comprehensive solution for Joomla communities.

Pager

Pager is a Joomla tool that streamlines the layout of database data for websites. It’s ideal for sites transitioning from native PHP with database-stored items or those with numerous products and technical specs. Instead of repetitive article creation, Pager uses a single PHP page template linked to product IDs in the database, eliminating redundancy and saving time. It features a text editor for uniform layout across products and a selection tool for database queries. Custom SQL queries are also supported, offering limitless possibilities for data presentation.

Photogallery for CB

Photo Gallery for Community Builder is a Joomla extension that enables users to create and manage personal image galleries with ease. The user-friendly frontend interface allows for simple navigation and organization of photos into categories. Features include a toolbar for image navigation, tagging system for identifying users in photos, and a comment management panel. It mirrors social media functionality with intuitive tagging and moderation tools, enhancing the community experience. The backend customization and optional component for administrators ensure control over content compliance.

Profile Ranking for CB

Profile Ranking for CB is a plugin that adds ‘like’ and ‘dislike’ buttons to Community Builder profiles, allowing users to vote on others’ profiles. Each user can cast one vote per profile, with the option to change it later. The system ensures that only logged-in users can vote and prevents self-voting, fostering an interactive community environment.

Search for K2

Search for K2 is a straightforward plugin for the K2 component in Joomla, designed to enhance article searching capabilities. After installation, it requires specifying a custom form created in K2, which will be displayed in the frontend search form. Activation is simple: once the plugin is enabled, users can immediately use the search feature within the K2 item list view, providing a seamless and efficient search experience.

Search source

Search Source is an indispensable Joomla extension for administrators, streamlining the search process within server files and databases. It’s not limited to file names; it delves into file contents and database fields for specified keywords. Ideal for editing components or adding HTML elements, it simplifies site management by providing precise locations of search terms, even in large sites, with searches typically under 30 seconds. Installation is straightforward, and results guide direct FTP modifications.

Secret text for kunena

Secret Text for Kunena is a straightforward plugin that adds a layer of privacy to forum discussions. With minimal configuration, it introduces a padlock button in the Kunena text editor. Clicking this button wraps the selected text within the [secret] tag. Text enclosed in the [secret] tag remains visible only to certain user groups, which can be defined in the backend. This feature is especially useful for private communications among forum moderators, ensuring that only users with special privileges can view the ‘secret’ messages.

Select to Social

Select to Social is a user-friendly plugin that enables quick and easy sharing of pages and text on your site. When text is selected with the mouse, a pop-up appears featuring customizable icons of popular social networks from the backend. Clicking on an icon, like Twitter’s, allows users to share not just the current page but also the highlighted text. Available icons include Twitter, Facebook, Google Plus, Pinterest, and LinkedIn, streamlining social sharing directly from your site.

We Hate Spam

We Hate Spam is a plugin that effectively eliminates spam during user registration by checking email addresses and IP addresses against a comprehensive database of known spammers. If a match is found, it blocks the registration process. It offers customization options, including notifications to administrators and personalized messages for blocked users. Additionally, it allows for the creation of white/blacklists of email addresses and is compatible with Community Builder.
